# countdown
Simple countdown timer. Written in Go.
```bash
curl -LO https://github.com/javimosch/countdown/releases/latest/download/countdown-linux-amd64
chmod +x countdown-linux-amd64; sudo mv countdown-linux-amd64 /usr/local/bin/countdown
```
Usage: `countdown 10` — counts down from 10 to 0

---

This tool is a plugin for [supercli](https://github.com/javimosch/supercli) — an AI-friendly, config-driven dynamic CLI platform.
